<objective>
Build `PublisherAliasResolver` — pipeline step [5] of the AppData orphan detection. It resolves common short forms (e.g., folder named `Adobe` matching `Adobe Inc.` in the registry, `JB` matching `JetBrains s.r.o.`, `NV` matching `NVIDIA Corporation`) into the canonical publisher name expected by `RegistryMatcher` (which already uses `FuzzyMatcher`). The resolver ships a curated `aliases.json` embedded resource and falls back to `FuzzyMatcher.ComputeMatch` when no alias declaration covers the input.

Purpose: The current `OrphanDetectorService.MatchesAnyProgram` (line 366-382) calls `FuzzyMatcher.IsMatch(node.Name, program.Publisher, program.DisplayName, FuzzyThreshold)`. On the corpus, this misses cases like the folder `Common Files\Adobe AIR` because no installed program is named `Adobe AIR` (the program is `Adobe Acrobat`) — the fuzzy match by token Jaccard fails. With aliases (`Adobe AIR` → canonical `Adobe Inc.`), the resolver returns Adobe and the registry matcher finds the existing Adobe Acrobat install. Goal: bridge real-world folder naming entropy without weakening the fuzzy threshold.
Output: 1 model + 1 interface + 1 implementation + 1 embedded JSON + ~9 tests. Suite remains green; new total ~160.
</objective>

```csharp
namespace DiskScout.Models;

/// <summary>
/// Maps a folder-name short form to the canonical publisher name as it appears in the
/// installed-programs registry. Used by the AppData orphan pipeline to bridge naming
/// entropy between filesystem folders and the registry's Uninstall keys.
/// </summary>
public sealed record PublisherAlias(
    string FolderName,        // e.g. "Adobe AIR", "MS", "JB", "NV"
    string CanonicalPublisher // e.g. "Adobe Inc.", "Microsoft Corporation", "JetBrains s.r.o.", "NVIDIA Corporation"
);
```

namespace DiskScout.Services;

    /// nor any fuzzy candidate matches.</summary>
    Task<string?> ResolveAsync(string folderName, IEnumerable<string> candidatePublishers, CancellationToken cancellationToken = default);

    Task LoadAsync(CancellationToken cancellationToken = default);
}
