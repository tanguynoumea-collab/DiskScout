<objective>
Build the **Install Tracker** service: while a third-party installer runs, capture FileSystem create/modify events (NTFS roots) and Registry write events (HKLM\SOFTWARE, HKCU\SOFTWARE, HKLM\SOFTWARE\WOW6432Node) into a deduplicated, persisted JSON trace. Each trace is later consumed by the wizard during uninstall to know exactly which files and registry keys to inspect for residue.

Purpose: Reach Revo Pro parity on tracked uninstalls — the trace is the ground truth of what an installer actually wrote.
Output: `IInstallTracker` + `IInstallTraceStore` + persisted JSON traces + tests.
</objective>

    Create `src/DiskScout.App/Services/InstallTracker.cs`. Class `public sealed class InstallTracker : IInstallTracker, IDisposable`.

    Constructor signature:
    ```csharp
    public InstallTracker(Serilog.ILogger logger, IInstallTraceStore store, IReadOnlyList<string>? roots = null, IReadOnlyList<string>? excludedPathSubstrings = null)
    ```

    Defaults when `roots == null`:
    - All `DriveInfo.GetDrives()` where `DriveType == DriveType.Fixed && IsReady == true`, mapped to `drive.RootDirectory.FullName`.

    Defaults when `excludedPathSubstrings == null`:
    ```
    ["\\$Recycle.Bin\\", "\\pagefile.sys", "\\hiberfil.sys", "\\swapfile.sys",
     "\\DiskScout\\diskscout.log", "\\AppData\\Local\\DiskScout\\install-traces\\",
     "\\AppData\\Local\\DiskScout\\quarantine\\"]
    ```

    Implementation rules:

    1. **FS watching:** one `FileSystemWatcher` per root with:
       - `IncludeSubdirectories = true`
       - `NotifyFilter = NotifyFilters.FileName | NotifyFilters.DirectoryName | NotifyFilters.LastWrite | NotifyFilters.CreationTime`
       - `InternalBufferSize = 65536` (max — 64 KB)
       - Subscribe to `Created` (-> `FileCreated`/`DirectoryCreated` based on `Directory.Exists`), `Changed` (-> `FileModified`)
       - Wrap event handlers in try/catch; on `InternalBufferOverflow` (`Error` event) log a warning at `Warning` level — do not throw.

    2. **Registry watching:** P/Invoke `RegNotifyChangeKeyValue` from `advapi32.dll`. Watch three keys:
       - `HKLM\SOFTWARE` (Registry64 view)
       - `HKLM\SOFTWARE\WOW6432Node` (Registry32 view)
       - `HKCU\SOFTWARE`

       For each, open via `RegistryKey.OpenBaseKey(...).OpenSubKey("SOFTWARE", writable: false)` (or `"SOFTWARE\\WOW6432Node"`), get the underlying handle via reflection on `SafeRegistryHandle Handle` property, call `RegNotifyChangeKeyValue(handle, watchSubtree: true, REG_NOTIFY_CHANGE_NAME | REG_NOTIFY_CHANGE_LAST_SET (=0x1|0x4), waitEvent, async: true)` on a Win32 manual-reset event. Spin a background `Task` per watched hive that loops `WaitHandle.WaitAny(new[] { winEvent, _stopEvent }, TimeSpan.FromMilliseconds(500))` and on signal records a `RegistryKeyCreated` (we cannot tell which subkey changed without re-enumeration — record the parent key path with kind `RegistryValueWritten`) and re-arms the notification.

       P/Invoke signature to add at top of file:
       ```csharp
       [DllImport("advapi32.dll", SetLastError = true)]
       private static extern int RegNotifyChangeKeyValue(
           IntPtr hKey, bool watchSubtree, int notifyFilter, IntPtr hEvent, bool asynchronous);

       private const int REG_NOTIFY_CHANGE_NAME    = 0x1;
       private const int REG_NOTIFY_CHANGE_LAST_SET = 0x4;
       ```

    3. **Deduplication:** maintain `HashSet<(InstallTraceEventKind, string)>` (case-insensitive on path). Only add to events list when the (kind, path) is new.

    4. **Filtering:** before adding any event, check `path` against `excludedPathSubstrings` using `IndexOf(StringComparison.OrdinalIgnoreCase)`. Skip if any substring matches.

    5. **StartAsync:** if `IsTracking == true`, throw `InvalidOperationException("Tracker already running.")`. Otherwise initialize watchers, set `IsTracking = true`, capture `_startedUtc = DateTime.UtcNow`, store the command-line/hint for the header.

    6. **StopAsync:** dispose all watchers and signal background tasks to exit (they are async so just flag a `_stopRequested = true` + Set the stop event). Wait up to 2 seconds via `Task.WhenAll(...).WaitAsync(TimeSpan.FromSeconds(2))`. Build the `InstallTrace`, save via `_store.SaveAsync(trace, cancellationToken)`, return the trace.

    7. **Tracker version:** hard-code `private const string TrackerVersion = "1.0";` and put in `InstallTraceHeader.TrackerVersion`.

    8. **Logging:** `_logger.Information("Install tracker started; roots={Roots}", string.Join(", ", _roots));` and `_logger.Information("Install tracker stopped; events={Count} duration={Sec}s", trace.Events.Count, (trace.Header.StoppedUtc - trace.Header.StartedUtc).TotalSeconds);`.

    9. **Tests** in `tests/DiskScout.Tests/InstallTrackerTests.cs`:
       - Use a temp dir + a fake `IInstallTraceStore` (in-memory `Mock<IInstallTraceStore>` recording the saved trace, OR a real `JsonInstallTraceStore` against a tempdir — choose Moq for purity)
       - Pass `roots: [tempDir]` to scope the test; `excludedPathSubstrings: []` to disable default filters for test 3 (then re-enable for test 3)
       - For test 2 use `HKCU\Software\DiskScoutTest_{Guid}` and clean it up in dispose
       - All tests must dispose the tracker; mark with `[Trait("Category", "Integration")]` if they touch the file system

    **Pitfall to avoid:** do NOT call `SafeRegistryHandle.DangerousGetHandle()` without acquiring it via `DangerousAddRef`/`Release`. Use `RegistryKey.Handle.DangerousAddRef`, capture the `IntPtr`, ensure `DangerousRelease` in dispose. Keep the `RegistryKey` instance alive for the lifetime of the watch.
